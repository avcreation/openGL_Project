Imac3 OpenGl Project
===============================

Build on Linux
--------------
<pre>
	make
</pre>

Launch
------
<pre>
	bin/debug/td3/td3 [-c] oui|non

	The "-c" option enable or disable the camera mouvement. To move the camera manually, press control and use the three buttons of the mouse.
</pre>

Needed Library
--------------
Boost, Assimp
